# Basic_Data_Visualization
Python Data Visualization with Matplotlib &amp; Seaborn

📊 Overview

This repository contains a collection of basic data visualization examples using Python.
The examples are implemented using two of the most widely used visualization libraries:

Matplotlib

Seaborn

The goal of this notebook is to provide a simple study reference for beginners who want to learn how to create common plots used in data analysis and exploratory data analysis (EDA).

Each plot includes a minimal example with code, making it easy for learners to understand and experiment.

🧰 Libraries Used
Matplotlib

Matplotlib is the core visualization library in Python.
It provides full control over plot creation and customization.

Seaborn

Seaborn is built on top of Matplotlib and provides a higher-level interface for creating statistical visualizations with cleaner aesthetics.

📈 Visualizations Included in This Notebook

The following plots and charts are demonstrated in this project.

1️⃣ Line Plot

Used to visualize trends over time or sequential data.

Implemented using:

plt.plot() (Matplotlib)

sns.lineplot() (Seaborn)

2️⃣ Bar Plot

Used to compare categorical values.

Implemented using:

plt.bar() (Matplotlib)

sns.barplot() (Seaborn)

3️⃣ Histogram

Used to visualize the distribution of numerical data.

Implemented using:

plt.hist() (Matplotlib)

sns.histplot() (Seaborn)

4️⃣ Scatter Plot

Used to analyze the relationship between two numerical variables.

Implemented using:

plt.scatter() (Matplotlib)

sns.scatterplot() (Seaborn)

5️⃣ Box Plot

Used to visualize data distribution and detect outliers.

Implemented using:

plt.boxplot() (Matplotlib)

sns.boxplot() (Seaborn)

6️⃣ Violin Plot

Shows data distribution along with density information.

Implemented using:

sns.violinplot() (Seaborn)

7️⃣ Heatmap

Used to visualize matrix data and correlations.

Implemented using:

plt.imshow() (Matplotlib)

sns.heatmap() (Seaborn)

8️⃣ Count Plot

Used to show the frequency of categorical values.

Implemented using:

sns.countplot() (Seaborn)

9️⃣ KDE Plot (Kernel Density Estimate)

Used to visualize the probability density distribution of a variable.

Implemented using:

sns.kdeplot() (Seaborn)

🔟 Pair Plot

Used for exploring relationships between multiple variables in a dataset.

Implemented using:

sns.pairplot()

Dataset used in the example:

Iris dataset

1️⃣1️⃣ Area Plot

Used to visualize filled trends over a continuous variable.

Implemented using:

plt.fill_between() (Matplotlib)

1️⃣2️⃣ Pie Chart

Used to represent proportions of categories in a dataset.

Implemented using:

plt.pie() (Matplotlib)
